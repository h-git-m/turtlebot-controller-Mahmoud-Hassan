1- Add "turtlebot_controller" package to your src folder in your worksapce
2- In your terminal, move to the workspace the directory ex: $ cd ../.. (this will move back twice inside folders)
3- $ source ~/.bashrc
4- Build only the package you developed $ colcon build --packages-select turtlebot_controller
5- $ source install/setup.bash
6- To check executables for this package, type: $ ros2 pkg executables turtlebot_controller You'll see
